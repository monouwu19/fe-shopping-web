# Synex React

## Chạy project

```bash
npm install
npm run dev
```

## Route
- `/dashboard`
- `/products`
- `/contact`
- `/cart`
- `/checkout`

## Ghi chú
- Đã chuyển các trang HTML sang React component.
- Trang `contact` và `checkout` đã dùng `useState` cho form.
- Trang `cart` đã có state tăng/giảm số lượng và xóa sản phẩm.
- Một số ảnh gốc như `ipad.png`, `macbook.png`, `airpods.png` không có trong source, nên đã tạm map sang ảnh placeholder để project chạy đủ.
